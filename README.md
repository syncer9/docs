### 폴더 구조 및 양식 예제
```
lehon05dev/data.git
│
├── 저장소(docs)
│    └── 대분류(App)
│          └── 중분류(Gitlab)
│                └── 소분류(Installation)
│                       └── Readme.md
│                       └── images(이미지 저장폴더)
└── 저장소(dev)
     └── 대분류(Shell)
           └── 소분류(bonding)
```


- 모든 문서는 docs 폴더에 분류별로 저장
- 소분류에는 반드시 Readme.md 파일생성 과 이미지 저장은 images에 상대 경로로 저장

- 모든 개발관련 스크립트 및 프로그램은 dev에 언어별로 저장(ex. Shell, Python, etc)
